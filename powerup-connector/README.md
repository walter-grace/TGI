# TGI Power-Up Connector

Minimal Trello iframe connector for API key registration. Deploy to get an HTTPS URL for the Trello Power-Up admin form.

## Deploy to Vercel

```bash
cd powerup-connector
npx vercel
```

Follow the prompts. Use the generated URL (e.g. `https://powerup-connector-xxx.vercel.app`) as the **Iframe connector URL** in [Trello Power-Up Admin](https://trello.com/power-ups/admin).
